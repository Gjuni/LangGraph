Langgraph의 시연 영상을 토대로 보안 뉴스의 게시글을 스크랩하고 보고서를 작성하는 에이전트 생성

URL: https://www.boannews.com/

역할: 날짜 별로 나온 사건사고를 정리하여 보고서를 작성한다.
    - 이에 따라 에이전트를 생성하는데 각 에이전트의 Tool은 Tools/{tool_name}.py 로 생성하여 각각의 파일로 생성하라.
    - 에이전트는 LangGraph로 만들어 노드별로 구조화하여 유지보수가 쉽게 사람이 수정이 가능하여야 한다.
    - 코드는 길지 않을 것 같다. 길게 작성하지 말고 컴팩트 있게 작성하라.

에이전트 정의: LangGraph의 에이전트와 Tool을 정의한다.
    - 사건사고 스크랩 에이전트: PlayWrite나 Croma처럼 웹사이트를 분석하는 에이전트
    - 사건사고 정리 에이전트: 가져온 정보를 토대로 정리하는 에이전트
    - 보고서 작성 에이전트: 정리된 정보를 토대로 보고서를 작성하는 에이전트
    - 이외의 필요시 더 생성.

폴더 구조: 각 폴더가 생성된다면 해당 MD 파일 처럼 정의하여 AI에게 명확한 역할을 부여할 수 있도록 한다.
    - 하네스 엔지니어링 기법으로 각 폴더의 명확한 내용을 정의하고 역할을 수행하라.
    - .env는 example_env를 만들어서 Token을 삽입할 수 있게 하라.

절대 하지 말아야할 규칙:
    - 코드는 길게 작성하지 않는다.
    - LangGraph 이외에 코드는 작성하지 않는다.
    - Tool로 정의된 것들은 반드시 Tools/의 하위 폴더로 들어가야 한다.
    - 서버 시간 기준이 아닌 보안 뉴스는 보고서에 들어가지 않는다.
    - 보고서의 요약 내용이 없으면 그건 보고서가 아니다. 보고서에 제목, 요약된 내용이 반드시 들어가야한다.

참고 사항: 영상 Ch 2.LangGraph 핵심 기능과 Ch 3. LangGraph 구조 설계를 참고하여 기능을 구현하라.
